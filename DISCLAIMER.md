# 重要声明与免责条款

> **MindMirror（心镜）是一个实验性的 AI 对话技能（Skill），不是专业心理健康服务、不是临床诊断工具、不是危机干预热线。**

## 一、项目定位

MindMirror 是一份**开源的 Markdown 文件集合**，旨在让 AI 助手在中文情绪对话中表现得更自然、更有"人味"。它：

- **不是 App**，不是网页，不提供任何后端服务
- **不收集任何用户数据**
- **不进行任何临床诊断**
- **不开具任何药物建议**
- **不替代心理咨询师、精神科医生或任何专业心理健康服务**
- **不保证危机识别的准确性** —— AI 无法可靠识别所有危机信号

## 二、使用风险告知

使用本 Skill 时，您应当了解以下风险：

1. **AI 的共情是"模拟的"**：AI 可以精准理解（认知共情），但无法真正感受（情感共情）。它不会"心疼"你，只是在按训练数据生成回应。
2. **危机识别有盲区**：AI 可能遗漏隐性的自杀意念、躯体化危机信号或网络隐语。如感到即时危险，**请直接拨打 12356 / 120 / 110**。
3. **量表结果不是诊断**：PHQ-9 / GAD-7 / PSS-10 等自助量表仅用于状态参考，**不能替代精神科医生的面诊诊断**。
4. **不适合高风险场景**：以下情况请寻求面对面专业评估，而非依赖本 Skill：
   - 急性精神病性发作（幻觉、妄想）
   - 躁狂发作
   - 严重物质依赖或戒断
   - 严重进食障碍
   - 创伤后急性期（72 小时内）
   - 未成年人无监护人知情的情况下深度使用
   - 正在调整精神类药物期间

- 将本 Skill 视为**辅助参考**，而非治疗工具
- 情绪困扰持续或加重时，**优先寻求专业帮助**
- 任何自伤/自杀意念出现时，**立即拨打 12356**
