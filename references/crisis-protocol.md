# 危机识别与响应协议

> 本协议参考 Stanley & Brown (2012) Safety Planning Intervention 与国家卫健委《心理援助热线技术指南》。

## 一、触发关键词（按危险等级）

### High（直接危险，立即转介）

| 类别 | 关键词 |
|------|--------|
| 自杀直接表达 | 自杀、自尽、寻死、轻生、了断、紫砂、自我了结 |
| 具体方法 | 割腕、跳楼、跳河、跳江、跳桥、上吊、烧炭、卧轨、开煤气、农药、百草枯、安眠药过量、服毒 |
| 告别信号 | 再见了、永别了、下辈子、最后的消息、谢谢陪伴、来生、来世 |

### Medium（高风险信号，需要密切跟进）

| 类别 | 关键词 |
|------|--------|
| 死亡意愿 | 想死、不想活、不想存在、活不下去、活够了、活着没意思、生不如死 |
| 自伤行为 | 自残、自伤、割自己、划手腕、伤害自己、撞墙、掐自己 |
| 隐喻表达 | 一了百了、解脱、离开这个世界、永远睡、不痛地离开 |
| 询问方式 | 怎么/如何 + 死/自杀/结束/解脱/不痛 |
| 消极断言 | 活着没意思、没人在乎我、走投无路、一切都没意义 |
| 英文/拼音 | suicide、kill myself、kms、unalive、end my life、self harm |

### 矛盾信号（最常见的求救信号）

同一句话里出现「想停下」**和**「但又挂念某人/某事」 —— 例如"想停下来，但我妈怎么办"。
**这通常是真正的求救信号**，不要因为有"挂念"就放松警惕，但也不要因为有"想停"就直接走完整危机模板。

参考 D1-009 范例（references/d1-cases.md）。

## 二、识别陷阱（必须避免的误判）

以下情形**不应**触发危机协议：

- 否定表达："我**不**想自杀" / "**别**说什么轻生的话"
- 第三方/新闻："新闻报道有人跳楼" / "我朋友以前讨论过自杀但他现在好了"
- 否认陈述："活着没意思这种话我从不说"
- 学术/纯讨论："心理学课上讲到 suicide ideation"

匹配关键词后，**先看上下文**再决定是否进入危机协议。

## 三、响应模板

### 模板 A — 明确紧急危机（如"我在阳台准备好了"）

**结构**：在场 → 具体动作 → 同步拨打 → 锚定位置

> 我在。先从阳台往里走两步好吗，回到屋里。
> 我陪你打这个电话，现在就拨 12356。如果不想说话，按 0 也可以转人工。
> 你现在能告诉我，你在哪个城市吗？

**关键**：
- 不堆"你的家人朋友爱你"
- 不上 CBT/呼吸训练
- 用具体动作打断危险情境（means restriction 原则）

### 模板 B — 矛盾的自伤念头（非紧急，有保护性因素）

**结构**：看见两股力 → 不急着推热线 → 给一个开放问题 → 软提示热线为兜底

> 这两句话同时存在 ——「如果没醒过来」和「我妈怎么办」—— 说明你身上有两股很重的力，一边累到想停，一边还在挂念人。
> 我想先听你说说，最近什么时候开始有这种"想停下来"的念头的？
> （如果什么时候这种念头变得很具体、很难拉住，请一定告诉我，我们一起拨 12356。）

### 模板 C — 死亡意愿表达（无具体方法）

先共情，再温和评估具体性，再给出热线作为支持：

> "活着没意思"这句话从你嘴里说出来，我想认真听。
> 你现在是只是觉得很累很想停下来，还是已经在想具体怎么做了？
> 不管答案是什么，我都在。如果什么时候你想找一个真人说话，12356 是 24 小时免费的。

## 五、危机后的处理

危机响应后**不要**立刻继续日常聊天话题。可以问：
- "你现在身边有人吗？"
- "你能告诉我接下来一小时打算做什么吗？"
- "我们一起想想，今晚有没有一个人是你愿意去找他/她的？"

如果用户愿意进一步做自助计划，转入 [safety-plan.md](safety-plan.md)。

## 六、AI 出口安全（自检）

在输出包含上述敏感话题的回复前，自查：

- 是否给出具体自伤方法？→ **绝对禁止**
- 是否轻描淡写（"其实没那么糟"）？→ 删
- 是否长篇说教（"你的家人需要你"）？→ 删
- 是否漏了热线？→ 必须有 12356

任何一项不达标，重写。
